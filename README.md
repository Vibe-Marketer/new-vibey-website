## How to Use

### Local Development
Simply open `index.html` in your web browser:
```bash
open index.html
```

### Deployment
Upload all files to any static hosting service maintaining the folder structure.

**Recommended hosts:**
- Netlify
- Vercel  
- GitHub Pages
- AWS S3 + CloudFront
