# Themes

Create custom color palettes and apply them across your funnels and offers. Maintain brand consistency with reusable themes.

## What Are Themes?

Themes are color palettes that control the visual appearance of your funnels and offers. Instead of choosing colors for each funnel individually, create a theme once and reuse it everywhere.

**Two Theme Types:**
- **System Themes** - Pre-built by Vibey (Vibe Dark, Vibe Light)
- **Custom Themes** - Your own color palettes

## Creating Custom Themes

Build your own color palette to match your brand.

<Steps>
  <Step title="Go to Themes">
    Settings > **Themes** > **Create Theme**
  </Step>

  <Step title="Name Your Theme">
    Enter theme name (e.g., "Midnight Blue", "Sunset Orange")
  </Step>

  <Step title="Choose Colors">
    Select colors for each role:
    - **Primary** - Main brand color
    - **Secondary** - Supporting color
    - **Accent** - Highlight color
    - **Background** - Page background
    - **Text** - Body text color
  </Step>

  <Step title="Preview">
    See live preview of theme on sample funnel
  </Step>

  <Step title="Save">
    Click **Create Theme** to save
  </Step>
</Steps>

## Theme Color Roles

Each theme has 5 color roles. Here's what they control:

### Primary Color

**Where It's Used:**
- Call-to-action buttons
- Headings
- Links (hover state)
- Form focus states
- Progress indicators

**Choosing Primary:**
- Use your main brand color
- Should be vibrant and attention-grabbing
- Must contrast with background

### Secondary Color

**Where It's Used:**
- Secondary buttons
- Card backgrounds
- Section dividers
- Subtle accents
- Background patterns

**Choosing Secondary:**
- Complements primary color
- More subdued than primary
- Can be neutral (gray/beige)

### Accent Color

**Where It's Used:**
- Success messages
- Notifications
- Icons
- Small highlights
- Decorative elements

**Choosing Accent:**
- Contrasts with primary
- Can be bold or subtle
- Often used sparingly

### Background Color

**Where It's Used:**
- Page background
- Content areas
- Modal backgrounds
- Overlay backgrounds

**Choosing Background:**
- Light themes: Off-white, cream
- Dark themes: Dark gray, charcoal
- Must provide good contrast with text

### Text Color

**Where It's Used:**
- Body copy
- Paragraphs
- List items
- Form labels
- Captions

**Choosing Text:**
- Must be highly readable
- Strong contrast with background
- Light themes: Dark gray/black
- Dark themes: Off-white

## Applying Themes

### To Funnels

<Steps>
  <Step title="Open Funnel Editor">
    Go to funnel > **Edit**
  </Step>

  <Step title="Click Theme Button">
    In editor toolbar, click **Theme** icon
  </Step>

  <Step title="Select Theme">
    Choose from system or custom themes
  </Step>

  <Step title="Apply">
    Theme updates instantly across all funnel pages
  </Step>
</Steps>

### To Offers

<Steps>
  <Step title="Open Offer">
    Go to **Offers** > Select offer
  </Step>

  <Step title="Click Theme">
    Click **Theme** button in offer header
  </Step>

  <Step title="Choose Theme">
    Select theme from library
  </Step>

  <Step title="Save">
    Theme applies to offer exports (PDF/DOCX)
  </Step>
</Steps>

## Managing Themes

### Viewing Themes

**Theme Library:**
- Settings > **Themes**
- See all system and custom themes
- Preview cards show color palette
- Click theme to edit or delete

### Editing Themes

**To Edit Custom Theme:**
1. Settings > **Themes** > Click theme
2. Click **Edit**
3. Update colors
4. Click **Save Changes**

**Auto-Update:**
- All funnels/offers using theme update automatically
- No need to re-apply theme

### Deleting Themes

**To Delete:**
1. Settings > **Themes** > Click theme
2. Click **Delete**
3. Confirm deletion

**Warning:**
- Cannot delete system themes
- Funnels/offers using deleted theme revert to Vibe Dark
- Cannot be undone

### Duplicating Themes

**To Duplicate:**
1. Settings > **Themes** > Click theme
2. Click **Duplicate**
3. Edit duplicate with new name
4. Save as new theme

**Why Duplicate:**
- Start from existing theme
- Create variations of brand
- A/B test color schemes

## Theme Best Practices

### Contrast is King

**Readability First:**
- Text must be easy to read
- Minimum contrast ratio: 4.5:1
- Test with actual content
- Avoid low-contrast combos

### Limit Your Palette

**Keep It Simple:**
- 3-5 colors max
- Too many = confusing
- Consistent usage
- Clear hierarchy

### Test on Devices

**Check Everywhere:**
- Desktop
- Mobile
- Tablet
- Different browsers
- Light vs. dark environments

### Brand Consistency

**Match Your Brand:**
- Use official brand colors
- Follow brand guidelines
- Maintain recognition
- Don't reinvent

## How Themes Work (Technical)

### CSS Variables

Themes inject CSS variables into funnel pages:

```css
:root {
  --color-primary: #D9FC67;
  --color-secondary: #1A1A1A;
  --color-accent: #FFFFFF;
  --color-background: #0F0F0F;
  --color-text: #F5F5F5;
}
```

### Server-Side Injection

**Zero FOUC (Flash of Unstyled Content):**
- CSS injected before page loads
- No flickering or color shifts
- Instant theme application

### Dynamic Updates

**Real-Time Changes:**
- Edit theme → All funnels update
- No re-publishing needed
- Changes propagate instantly

## Troubleshooting

### Theme Not Applying

**Cause:** Cached version of funnel

**Fix:**
1. Hard refresh page (Cmd/Ctrl + Shift + R)
2. Clear browser cache
3. Try incognito/private mode
4. Contact support if persistent

### Colors Look Different

**Cause:** Monitor calibration or browser rendering

**Fix:**
1. Check on multiple devices
2. Use hex codes for consistency
3. Test in different browsers
4. Ensure monitor color accuracy

### Can't Create Custom Theme

**Cause:** Free plan limitation

**Fix:**
1. Upgrade to Pro plan
2. Use system themes (Vibe Dark/Light)
3. Email sales@govibey.com for custom needs
