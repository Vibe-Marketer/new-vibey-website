# Campaign Map

Visualize and manage your entire marketing campaign as an interactive flow diagram. See how funnels, sequences, and content work together.

<Info>
  Campaign Map is a **Pro-only feature** available for all campaigns.
</Info>

## What Is Campaign Map?

Campaign Map gives you a bird's-eye view of your entire marketing campaign. Instead of managing funnels and sequences separately, see everything connected in one visual workspace.

**Key Features:**
- **Interactive Flow Diagram** - Drag-and-drop nodes, zoom, pan
- **Real-Time Preview** - Click any node to preview content
- **Auto-Layout** - Intelligent node positioning
- **Full-Screen Mode** - Expand for detailed view
- **Live Editing** - Add, remove, and connect components

---

## How It Works

### Opening Campaign Map

<Steps>
  <Step title="Go to Campaign">
    Navigate to **Campaigns** > Select your campaign
  </Step>

  <Step title="Click Map">
    Click **View Map** button in campaign header
  </Step>

  <Step title="Explore">
    Interactive map slides in from right (60-70% width)
  </Step>
</Steps>

---

## Map Elements

### Node Types

**Funnel Nodes**
- Visual preview of funnel pages
- Click to view full funnel details
- Shows connected sequences
- Edit funnel directly from map

**Sequence Nodes**
- Email sequence overview
- Shows number of emails
- Preview first email
- Expand to see all steps

**Sequence Step Nodes**
- Individual email preview
- Subject line visible
- Body content preview
- Edit email inline

**Add Component Nodes**
- Plus (+) button placeholders
- Click to add funnel or sequence
- Auto-connects to campaign flow

## Map Controls

### Toolbar Actions

**View Controls:**
- **Zoom In** - Closer view of nodes
- **Zoom Out** - See more of canvas
- **Fit View** - Auto-fit all nodes
- **Mini-Map** - Toggle mini-map overlay
- **Full Screen** - Expand to full window

**Layout Controls:**
- **Auto-Layout** - Reorganize nodes automatically
- **Show/Hide Nodes** - Toggle node visibility
- **Restore Hidden** - Bring back hidden nodes

**Edit Controls:**
- **Delete Node** - Remove component from campaign
- **Disconnect** - Remove connection between nodes
- **Add Component** - Insert new funnel/sequence

## Using Campaign Map

### Viewing Your Campaign Flow

1. **Open map** to see complete campaign structure
2. **Follow arrows** to understand customer journey
3. **Click nodes** to preview content
4. **Zoom/pan** to navigate large campaigns

### Adding Components

<Steps>
  <Step title="Click Plus Node">
    Click any **+ Add Component** node
  </Step>

  <Step title="Choose Type">
    Select **Add Funnel** or **Add Sequence**
  </Step>

  <Step title="Configure">
    Fill in component details
  </Step>

  <Step title="Generate">
    AI generates content and adds to map
  </Step>
</Steps>

### Editing Nodes

**To Edit Content:**
1. Click node to open preview panel
2. Click **Edit** in preview
3. Make changes in editor
4. Changes reflect immediately

**To Remove Node:**
1. Click node to select
2. Press **Delete** key or click trash icon
3. Confirm removal
4. Node and connections removed

### Reorganizing Layout

**Manual Layout:**
- Drag nodes to new positions
- Connections update automatically
- Layout persists when you return

**Auto-Layout:**
- Click **Auto-Layout** button
- Algorithm positions nodes optimally
- Vertical flow from top to bottom
- Evenly spaced and aligned

## Map Features

### Email Preview Panel

Click any sequence step to see:
- **Subject Line** - Email subject
- **Preview Text** - First 100 characters
- **Full Body** - Complete email content
- **Send Timing** - Delay from previous email
- **Edit Button** - Open email editor

### Mini-Map

**What It Shows:**
- Overview of entire canvas
- Your current viewport (highlighted box)
- All nodes (small dots)

**How to Use:**
- Toggle on/off with mini-map button
- Click mini-map to jump to location
- Drag viewport box to pan

### Full-Screen Mode

**Benefits:**
- More canvas space
- Better for complex campaigns
- Easier to see connections
- Less distraction

**Controls:**
- Press **F** or click full-screen icon
- Press **Esc** to exit
- All editing works in full-screen

## Campaign Flow Patterns

### Linear Campaign

```
Funnel → Sequence → Funnel → Sequence
```

**Best For:**
- Simple campaigns
- Single offer
- Clear path to conversion

### Multi-Path Campaign

```
         ↗ Sequence A → Funnel A
Funnel →  Sequence B → Funnel B
         ↘ Sequence C → Funnel C
```

**Best For:**
- Segmented audiences
- Multiple offers
- A/B testing

### Webinar Campaign

```
Registration Funnel → Reminder Sequence → Live Funnel → Replay Sequence → Offer Funnel
```

**Best For:**
- Webinar launches
- Multi-step education
- High-ticket offers

## Map Limitations

**Current Limits:**
- Up to 50 nodes per campaign
- Auto-layout works best with <20 nodes
- No cross-campaign connections
- Cannot merge campaigns in map

**Coming Soon:**

## Tips for Using Campaign Map

### Keep It Simple

**Don't Overcomplicate:**
- Start with 3-5 components
- Add more as needed
- Too many nodes = confusing

### Use Descriptive Names

**Name Components Clearly:**
- "Lead Magnet Funnel" vs. "Funnel 1"
- "Welcome Sequence" vs. "Sequence A"
- Easier to navigate map

### Review Regularly

**Check Your Flow:**
- Is the path clear?
- Are there gaps?
- Does timing make sense?
- Any dead ends?

### Test Before Launch

**Validate Flow:**
- Walk through customer journey
- Preview each component
- Check email timing
- Verify all links work

## Troubleshooting

### Map Not Loading

**Cause:** Browser compatibility or large campaign

**Fix:**
1. Refresh page
2. Try different browser (Chrome recommended)
3. Clear browser cache
4. Contact support if issue persists

### Nodes Overlapping

**Cause:** Manual positioning or too many nodes

**Fix:**
1. Click **Auto-Layout** to reorganize
2. Zoom out to see full canvas
3. Manually drag nodes apart
4. Consider splitting into multiple campaigns

### Preview Not Showing

**Cause:** Content not yet generated

**Fix:**
1. Check if component has finished generating
2. Refresh campaign map
3. Try clicking different node
4. Contact support if persistent
